About me

About me

Kişisel Geçmiş:

Ben İrem Can. 12 Kasım 2002 tarihinde İstanbul'da doğdum. Süleyman Demirel Anadolu lisesinde mezun oldum. Eğitim hayatıma Bandırma On Yedi Eylül Üniversitesi'nde devam ediyorum. Bilgisayar Mühendisliği 3. sınıf öğrencisiyim. Yazılım Mühendisliği programı ile de çift anadal yapmaktayım.

Projelerim:

Web projeleri için Java ve Spring Boot framework kullanarak Rest Api geliştiriyorum. Spring ile birlikte Jpa, Hibernate, Mvc , Postgresql ve Elasticsearch biliyorum. Projelerini aktif olarak kullanıyorum.Node.js, Python ve mikro hizmetler mimarisinde uzmanım. PostgreSQL, MySQL ve Redis gibi veritabanlarını kullanarak Nodejs, NestJS-Expresjs, FastAPI, Django ve Flask ile ölçeklenebilir web uygulamaları geliştirdim. Git akış stratejilerinde ve kod kalitesi ve sürdürülebilirlik için Jest ile birim testleri yazma konusunda uzmanım. Grafana, Prometheus ve Jaeger kullanarak performans optimizasyonu konusunda deneyimim var.Ayrıca, Postman kullanarak üçüncü taraf API'leri entegre etme ve test etme, Docker ve AWS gibi platformlara uygulama dağıtma ve gerçek zamanlı bildirimler için web kancaları ve web yuvaları uygulama konusunda uzmanlığa sahibim. Verimli ve zamanında geliştirme için çevik metodolojiler kullandım ve yüksek kaliteli sonuçlar elde ettim.Ayrıca dinamik ve duyarlı kullanıcı arayüzleri için ReactJS kullanan ve ön uç ile arka uç arasında sorunsuz iletişim sağlamak için Node.js, Python ve mikro hizmetler mimarisi gibi sunucu tarafı teknolojileri içeren tam kapsamlı web geliştirme deneyimim var.
